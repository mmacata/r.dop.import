DESCRIPTION

r.get.dop downloads digital orthophotos within a specified area in Germany. This is made possible by using the available geodata by the German federal states. The downloaded digital orthophots will be build up to one virtual raster in GRASS GIS. As input area of interest you can use an existing vector map in GRASS GIS. For example you can use v.check.bundesland for checking the federal state where the area of interest is located and then use that output in r.dop.import to import the appropriate digital orthophotos.

SEE ALSO

r.buildvrt, r.import, v.check.bundesland
