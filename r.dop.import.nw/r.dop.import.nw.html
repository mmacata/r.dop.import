DESCRIPTION

r.dop.import.nw downloads digital orthophotos (DOP) for Nordrhein-Westfalen (NW) and area of interest.

EXAMPLE

Import DOP

Import DOPs with native resolution: 
r.dop.import.nw aoi=aoi_NW output=dop_NW -r
