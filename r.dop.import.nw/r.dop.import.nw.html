DESCRIPTION

r.dop.import.nw downloads and imports digital orthophotos (DOP) for Nordrhein-Westfalen (NW) and area of interest. The data "Digitale Orthophotos" is downloaded from Geoportal Nordrhein-Westfalen and can be used under the specification of the license: Â© Geobasis NRW, Datenlizenz Deutschland - Zero - Version 2.0, DOP.

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.nw aoi=aoi_NW output=dop_NW -r
