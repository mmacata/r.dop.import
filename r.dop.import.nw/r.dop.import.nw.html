DESCRIPTION

r.dop.import.nw downloads and imports digital orthophotos (DOP) for Nordrhein-Westfalen (NW) and area of interest.

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.nw aoi=aoi_NW output=dop_NW -r
