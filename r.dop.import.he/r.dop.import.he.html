DESCRIPTION

r.dop.import.he downloads and imports digital orthophotos (DOP) for Hessen (HE) and area of interest using the respective WMS.

The data "Digitale Orthophotos" can be used under the license:
Geobasisdaten © Hessische Verwaltung für Bodenmanagement und Geoinformation: Digitale Orthophotos, Datenlizenz Deutschland - Zero - Version 2.0

EXAMPLES

Import DOPs

Import DOPs with native resolution: 
r.dop.import.he aoi=aoi_HE output=dop_HE -r
